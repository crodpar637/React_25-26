(async, inner) generatePDF(zonaImpresion, nombreDocumento) → {void}

Genera un archivo PDF a partir del contenido de un elemento HTML Proceso: 1. Captura el elemento HTML especificado usando html2canvas 2. Convierte el canvas a imagen PNG 3. Crea un documento PDF en formato A4 4. Ajusta la altura proporcional del contenido 5. Agrega la imagen al PDF 6. Descarga el archivo con el nombre especificado
