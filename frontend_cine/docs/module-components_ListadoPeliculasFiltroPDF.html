Module: components/ListadoPeliculasFiltroPDF

Componente PDF para exportar listado de películas filtradas Componente React-PDF que genera un documento PDF con tabla de películas. Utiliza StyleSheet para estilizar tabla de 5 columnas con bordes, encabezados con color de fondo gris y truncamiento de sinopsis.
